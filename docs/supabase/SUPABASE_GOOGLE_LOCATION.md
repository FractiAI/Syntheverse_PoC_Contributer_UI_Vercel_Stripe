## Exact Navigation Path:

1. supabase.com ‚Üí Sign in
2. Select your project
3. Left sidebar ‚Üí 'Authentication'
4. Top tabs ‚Üí Click 'Providers'
5. Scroll down to find 'Google'

## If you don't see 'Providers' tab:

- You might be on the old Supabase dashboard
- Try refreshing the page
- Or check if you're in the right project

## If Google is missing:

- Check your Supabase plan (free tier supports OAuth)
- Try logging out and back in
- Contact Supabase support if persistent

## Alternative: Check URL

Your URL should be:
https://app.supabase.io/project/YOUR_PROJECT/auth/providers

Replace YOUR_PROJECT with your actual project name.
